## About
  A rating system to rate and review your teacher, place to know and learn from best teachers and pick the best ones.

## Tech Stack
 - [React](https://reactjs.org/)
 - [Node Js ](https://nodejs.org/en/) and [Express Js](https://expressjs.com/)
